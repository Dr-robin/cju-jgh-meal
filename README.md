# cju-jgh-meal
Jeju Jungang Girl's High School Meal Parser

## Usage
For example: http://junganggh.planet.moe/2017/3/20/lunch
/yyyy/m/d/(lunch|dinner)

## Response
For example: {"cal":"801.4Kcal","meal":["잡곡밥","북어포콩나물국","돼지고기자장볶음","무초절임쌈","애배추느타리무침","배추김치"]}
